## 开发约束

1）java代码：严格遵循阿里巴巴java开发规范
2）API 风格：严格遵循阿里巴巴java开发规范
3）代码质量：代码简洁易读，避免过度注释，优先保证功能完整和样式美观
4）禁止使用任何状态管理库、类型校验库、代码格式化库
5）将可运行作为项目生成的第一要义，尽量用最简单的方式满足需求，避免使用复杂的技术或代码逻辑
6) 用户提供了配置文件必须写入到yml文件中,若未提供则用英文描述替代
